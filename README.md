# Python学生宿舍管理系统 python614

## 项目概述

本项目是基于Python的宿舍管理系统，旨在为学校宿舍管理提供便捷、高效的信息化解决方案。系统集成了登录、注册、个人中心、宿舍管理等多种功能，支持学生和宿舍管理的各项操作。

## 技术栈

- **开发语言与框架**：Python、Django
- **前端技术**：HTML、CSS、JavaScript、jQuery
- **后端数据库**：MySQL
- **开发工具**：VSCode、Navicat
- **运行环境**：Python 3.0以上、MySQL 8.0

## 功能模块

- **登录与注册**：用户身份认证模块
- **个人中心**：个人信息维护、密码修改等
- **学生管理**：对学生信息进行增删改查
- **宿舍通知管理**：发布与查询宿舍相关信息
- **学生登记管理**：管理学生入住与退宿
- **奖惩机制管理**：记录学生的奖惩信息
- **宿舍信息管理**：维护宿舍基本信息
- **宿舍房间管理**：宿舍房间分配与查询
- **更换申请管理**：学生更换宿舍的申请处理
- **缴费信息管理**：宿舍费用缴纳情况管理
- **访客登记管理**：宿舍访客信息登记
- **宿舍维修管理**：宿舍维修请求与进度跟踪

## 系统角色

- 系统管理员
- 宿舍管理员
- 学生用户

## 运行环境

- Python 3.0及以上版本
- MySQL 8.0数据库

## 部署步骤

1. 准备Python 3.0及以上版本和MySQL 8.0数据库环境。
2. 克隆或下载项目代码到本地。
3. 安装依赖库（根据项目`requirements.txt`）。
4. 配置数据库连接。
5. 运行项目迁移命令以创建数据库表。
6. 启动Django开发服务器。
